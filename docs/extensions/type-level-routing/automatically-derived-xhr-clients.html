Automatically Derived XHR Clients¶

As we represent routing as types, we can derive XHR clients from those types. In a client-side application, for example one written using Pux or Halogen, we use the purescript-hyper-routing-xhr library to derive such client functions. We get functions with type-safe parameters and encoding/decoding, using our routed types, and results in the Aff monad, which are easily integrated into most frameworks.
